# NYU GenAI Executive Deck

This folder is a repo-ready static site for the executive presentation.

Files:
- `index.html`: self-contained presentation deck
- `.nojekyll`: disables Jekyll processing for GitHub Pages

To publish on GitHub Pages:
1. Create a new repository.
2. Push the contents of this folder to the `main` branch.
3. Enable Pages from the repository root.

The presentation is self-contained, so no extra assets are required.
